PROJECT TITLE: Hermione Benitez's Portfolio Website

SHORT DESCRIPTION:
- This portfolio website contains all the information about Hermione Benitez, including her basic information, contact information, her skills, achievements, and experiences that could help the
audiences dig deeper into her qualities. This would help the clients know who are they going to work with and could give them a valuable insights about her works.

HOW TO RUN LOCALLY?
- To run this website in your local devices, download the ZIP file in this repository and unzip it in your file storage to unwrap all the assets.
- You must firstly install the IIS Server to host the web application built in ASP.NET or run it using https.
- Run the solution file, either through Visual Studio Community (preferably) or Visual Studio Code.
